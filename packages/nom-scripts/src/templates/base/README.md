## Scripts

### Start

Builds your app in development mode. You will be reported of any errors with your code.

```sh
yarn start
```

or with npm:

```sh
npm start
```

### Test

Runs unit tests.

```sh
yarn test
```

or with npm:

```sh
npm test
```

### Build

Builds your app for use in production environments.

```sh
yarn build
```

or with npm

```sh
npm run build
```
