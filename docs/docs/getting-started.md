## Quick Start

You can run Create Nom App with any of `npm`, `yarn`, or `npx`.

<!--DOCUSAURUS_CODE_TABS-->
<!--npx-->
```sh
npx create-nom-app hello-world
```

<!--yarn-->
```sh
yarn create nom-app hello-world
```

<!--npm-->
```sh
npm init nom-app hello-world
```

<!--END_DOCUSAURUS_CODE_TABS-->
