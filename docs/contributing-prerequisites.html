Contributor Prerequisites

Before your begin, make sure you have all the prerequisite software on your system.

  • Git
  • Docker & Docker Compose
  • Yarn
  • Node.JS, LTS version >=10.15.1, < 11.0.0 or LTS/dubnium recommended.
  • Make

Note: Docker & Docker Compose may not be required if you only intend to make changes on docs/ or packages/*.
